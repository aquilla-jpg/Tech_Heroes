v. Purpose of the code: The purpose of this code is to create a Point of Sales System. This system allows cashiers to add items to a shopping cart, remove items, view the cart, calculate the total bill (including tax), accept payment, and generate a receipt.
vi. How to run the code: To run the code, ensure you have the following:
				1. A notebook (Google Colab, Jupyter Notebook, or PyCharm)
				2. Remove the program from the zip folder
				3. If you are using PyCharm, follow steps 4 - 5
				4. Open PyCharm, select " Open file", and look for the program
				5. Ensure that you have pandas installed as a module
				6. For Colab, go to the upload section, click on browse, and find the program to open it
				7. For Jupyter, navigate to where you saved the program
				8. Press the play button or the run all button
				9. Follow the instructions on the screen 
vii. Assumptions: The program assumes that stock quantities are updated only during the sessions. Inventory resets when the program restarts unless manually saved 
viii. Limitations: Stock data is not permanently saved. Once the program is closed, updated stock quantities are lost. 
		   The program only accepts cash payment. 
		   No receipt file is exported.
